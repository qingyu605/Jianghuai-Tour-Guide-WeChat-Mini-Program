# 江淮风景线 - 微信小程序

## 项目简介

**江淮风景线**是一款专为江淮地区（安徽、江苏部分地区）打造的旅游导览微信小程序，提供景点信息查询、旅游路线推荐、语音导览等功能，为游客提供便捷、全面的江淮地区旅游信息。

## 功能特点

- 📍 **景点导览**：收录黄山、九华山、宏村、夫子庙、中山陵、瘦西湖等多个5A级景区
- 🗺️ **旅游路线**：设计多条特色旅游路线，满足不同游客需求
- 🎧 **语音导览**：提供景点语音讲解服务
- 📱 **位置服务**：支持基于位置的景点推荐
- 📷 **景点相册**：展示景点多张精美图片
- 💾 **收藏功能**：支持收藏喜欢的景点

## 技术栈

- 微信小程序开发
- JavaScript
- WXML/WXSS
- 微信小程序API

## 目录结构


## 如何运行

1. 克隆或下载本项目到本地
2. 打开微信开发者工具
3. 导入项目，选择项目根目录
4. 填写AppID（若没有可选择"测试号"）
5. 点击"编译"按钮运行项目

## 数据说明

项目数据存储在 `utils/data.js` 文件中，包含：

- **景点数据**：包括景点名称、图片、位置、分类、等级、描述、坐标、开放时间、门票价格、语音导览链接等
- **路线数据**：包括路线名称、天数、图片、描述、包含景点、行程安排等

## 主要页面

1. **首页**：轮播图展示、热门景点推荐
2. **景点**：景点列表、分类筛选、搜索功能
3. **路线**：推荐旅游路线、详细行程安排
4. **我的**：个人信息、收藏管理
